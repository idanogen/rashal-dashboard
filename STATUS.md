# STATUS — rashal-dashboard (לקוח: ראש"ל)
עודכן: 2026-06-23

## סיכום
מערכת לניהול ההזמנות והמשלוחים של הלקוח ראש"ל, כולל מפה עם מסלולי נסיעה והתראות וואטסאפ. עובדת. ממתינים לחיבור הסופי של הוואטסאפ.

## מה זה
דשבורד הזמנות/משלוחים ל-ראש"ל. React 19 + TypeScript + Supabase.
אינטגרציית WhatsApp דרך heyy.io (סטאק דמו+אמיתי בנוי).

## איפה זה עומד (אחרון: 2026-06-21)
- מפה: קו מסלול עוקב אחרי כבישים + מרחק/זמן (directions service).
- geocoding עבר מ-Nominatim לשירות המרכזי `ogen-geo-service`.
- חיפוש לפי שם/מספר לקוח/טלפון במשלוחים+שירות + חיווי "כבר טופל".

## מה הבא
- מיפוי Make: Priority CUSTNAME → customer_number (עדיין TODO).

## פתוח / ממתין
- אינטגרציית heyy.io: ממתינה לחשבון heyy + אישור תבנית Meta.

## Gotchas (Supabase)
- env vars ב-Vercel עם `\n` נגרר שובר realtime+auth; מפתחות JWT הוסבו ל-HS256; שמות עברית → email סינתטי מ-hash.
